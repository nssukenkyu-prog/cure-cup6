# CURE CUP 2026 - 10周年記念ピックルボール大会

日本体育大学スポーツキュアセンター横浜・健志台接骨院の開院10周年を記念したピックルボール大会「キュアカップ2026」の公式ホームページです。

## 🎯 プロジェクト概要

このウェブサイトは、2026年2月22日に開催される記念イベントの告知、協賛企業の募集、および参加者の募集を目的として作成されました。

### イベント詳細

- **大会名**: CURE CUP 2026
- **開催日時**: 2026年2月22日（日）14:00〜18:00（予定）
- **会場**: 日本体育大学 横浜・健志台キャンパス 米本記念体育館
- **参加資格**: 年齢・性別・経験問わず
- **定員**: 48名 / 24ペア（先着順）
- **参加費**: ペア申込み ¥1,000/人、個人申込み ¥1,500/人

## 🌐 ウェブサイトの公開方法

このウェブサイトを公開（デプロイ）するには、以下の手順に従ってください。

### 方法1: Publishタブから公開（推奨・最も簡単）

1. **画面上部の「Publish」タブをクリック**
2. **「Publish」ボタンをクリック**
3. **自動的にデプロイが開始されます**
4. **デプロイ完了後、公開URLが表示されます**
   - 例: `https://your-site.gensparksite.com`
5. **このURLを共有することで、誰でもアクセス可能になります**

### 方法2: 他のホスティングサービスを利用

#### Netlify（無料、推奨）
1. プロジェクトファイルをすべてダウンロード
2. [Netlify](https://www.netlify.com/)にアクセス
3. 「Add new site」→「Deploy manually」を選択
4. プロジェクトフォルダをドラッグ&ドロップ
5. 自動的にデプロイされ、URLが発行されます

#### GitHub Pages（無料）
1. GitHubアカウントを作成
2. 新しいリポジトリを作成
3. プロジェクトファイルをアップロード
4. Settings → Pages → Sourceを「main」ブランチに設定
5. `https://username.github.io/repository-name/` でアクセス可能

#### Vercel（無料）
1. [Vercel](https://vercel.com/)にアクセス
2. 「Add New Project」を選択
3. プロジェクトファイルをインポート
4. 自動的にデプロイされます

### 公開後のカスタマイズ

#### カスタムドメインの設定
- 独自ドメイン（例: curecup2026.com）を取得している場合
- 各ホスティングサービスの設定画面でドメインを追加可能

#### SSL証明書（HTTPS）
- Netlify、Vercel、GitHub Pagesは自動的にSSL証明書を発行
- 「https://」で安全にアクセス可能
